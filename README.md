# 📚 City Bookshop Management System

This project is a **Bookshop Management System** developed as a **team project for university coursework**.

The system helps manage bookshop operations such as handling books, customers, and orders in an organized way.

---

## 🎯 Project Objective
The objective of this project is to develop a system that simplifies the management of a bookshop by maintaining records of books, customers, and sales.

---

## 🛠 Technologies Used
- Java
- PHP
- MySQL
- HTML
- CSS
- JavaScript

---

## ✨ Features
- Add, update, and delete books
- Manage customer information
- Record and manage book orders
- Search books easily
- Simple user interface
